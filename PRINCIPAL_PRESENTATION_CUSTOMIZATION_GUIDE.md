## 📸 HOW TO ADD IMAGES IN POWERPOINT

### Method 1: Quick Insert
```
1. Open SMAP_Principal_Presentation_Enhanced.pptx
2. Click on slide where you want image
3. Click Insert menu → Pictures → From this device
4. Select image file from computer
5. Click Insert
6. Resize by dragging corners (hold Shift to keep proportions)
7. Move to desired position
```

### Method 2: Copy-Paste
```
1. Have image open in file explorer or browser
2. Right-click image → Copy
3. Click on slide in PowerPoint
4. Right-click → Paste
5. Resize and position as needed
```

### Method 3: Insert from File
```
1. Click Insert → Pictures
2. Browse to image file
3. Select and Insert
4. Adjust size (Width: 2-3 inches typical for team photos)
```

**Best Practices:**
- All images should be high-quality (300 DPI or higher)
- Keep consistent sizing for team photos
- Leave text readable (don't cover text with images)
- Use white or light backgrounds to contrast with dark blue header
